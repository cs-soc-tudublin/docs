# **Golem**

Golem is our primary Reverse Proxy for all web traffic.

Ports 80 & 443 are port-forwarded to Golem through the router.

NGINX is used as the Reverse Proxy for all HTTP(S) traffic.

**IP:** 10.0.0.110

| **Item**      | **Spec** |
| ------------- | -------- |
| **Sockets**   | 1        |
| **Cores**     | 1        |
| **RAM**       | 2GiB     |
| **Hard Disk** | 32GB     |
| **HA?**       | Yes      |

In memoriam, Golem Soc.
